
🚀 CONSULTA DE PEÇAS - INSTRUÇÕES LOCAIS

1. Verifique se o Python está instalado
2. Instale as bibliotecas necessárias:

    py -m pip install streamlit pandas openpyxl

3. Rode o app:

    py -m streamlit run app_streamlit.py --server.port 8502

4. O navegador abrirá com o sistema

- Clique em "🔄 Recarregar banco de dados" no app
